QU'EST CE QUE LE REFERENCEMENT INTERNET ?

Agissant sur le positionnement d'un site dans les résultats des moteurs de recherche, le référencement internet est indispensable pour être trouvé facilement. Différentes actions sont possibles afin d'améliorer le référencement d'un site Internet : le référencement technique SEO et le référencement optimisé SEM.

LE REFERENCEMENT SOCIAL

Egalement appelé community management, le référencement social a pour but de profiter des réseaux sociaux pour faire connaître une marque, une société, un produit. Le community manager doit agir quotidiennement sur les réseaux notamment en correspondant avec les internautes.

LE REFERENCEMENT MARKETING

Complémentaire au naturel, le référencement marketing consiste en des actions publicitaires permettant de générer un flux de visiteurs qualifiés vers le site Internet dans le but de déclencher des ventes ou des prises de contact.

E-REPUTATION

Entreprises, personnalités, particuliers, ... le web conserve toutes vos données. Informations obsolètes, actualités dépassées, avis négatifs, il convient aujourd'hui de surveiller activement tout ce qu'il peut se dire sur les réseaux sociaux, les blogs, les forums ou encore les sites d'opinion.
